# MetalCharts — Public Showcase
Public repository showcasing the MetalCharts project.

🌐 Website: https://metalcharts.net

MetalCharts is an independent project providing music charts dedicated to Metal.
The site features artist rankings, album charts, country/label/genre breakdowns,
and an advanced sorting module for all charted albums.

All data used is public (artist names, albums, countries, labels, release dates)
and enriched with internal calculations (points, cumulative scores, best ranks,
weeks charted, etc.).

## Main Features

- Album rankings (MetalCharts)
- Artist rankings (ArtistCharts)
- Album sorting module (Trialbums)
- Country rankings (CountryCharts)
- Label rankings (LabelCharts)
- Genre rankings (GenreCharts)
- New entries in the weekly Top 100 (NewEntries)
- New videos from charted artists (NewVideos)
- Hall of Fame: rotation of albums that reached #1
- Multilingual interface: EN, FR, DE, PT, ES
- Advanced autocomplete (artist search)
- Name normalization (slug) compatible with image filenames
- Static site optimized for speed and responsiveness

## Technologies

### Frontend
- Astro
- HTML5
- CSS3
- Vanilla JavaScript

### Backend / Tools
- C# (BuildXXX scripts generating JSON files)

### Data
- Static JSON files
- Internal calculations (points, scores, best rank, etc.)

### Deployment
- Netlify

## Main Pages

- **MetalCharts (Hall of Fame)** — rotation of #1 albums
- **ArtistCharts** — artist rankings
- **MetalCharts (Albums)** — weekly album chart
- **Trialbums** — album sorting module
- **CountryCharts** — country rankings
- **LabelCharts** — label rankings
- **GenreCharts** — genre rankings
- **NewEntries** — new albums entering the Top 100
- **NewVideos** — new videos from charted artists
- **About** — project presentation
- **Contact** — coming soon

## Project Goals

- Provide a modern alternative to traditional music charts
- Offer a fast, clear and accessible interface
- Make data easy to browse, sort and explore
- Build an ecosystem around MetalCharts

# MetalCharts — Présentation publique
Dépôt public présentant le projet MetalCharts.

🌐 Site web : https://metalcharts.net

MetalCharts est un projet indépendant proposant des classements musicaux
spécialisés dans le Metal. Le site présente des charts d’artistes, d’albums,
des analyses par pays, labels, genres, ainsi qu’un module avancé de tri des
albums classés.

Les données utilisées sont exclusivement publiques (noms d’artistes, albums,
pays, labels, dates de sortie) et sont enrichies par des calculs internes
(points, scores cumulés, meilleures positions, semaines classées, etc.).

## Fonctionnalités principales

- Classements d’albums (MetalCharts)
- Classements d’artistes (ArtistCharts)
- Module de tri des albums (Trialbums)
- Classements par pays (CountryCharts)
- Classements par labels (LabelCharts)
- Classements par genres (GenreCharts)
- Nouveaux albums entrés dans le Top 100 (NewEntries)
- Nouvelles vidéos publiées par les artistes classés (NewVideos)
- Hall of Fame : rotation des albums ayant atteint la 1ère place
- Interface multilingue : FR, DE, PT, ES, EN
- Autocomplétion avancée (recherche d’artistes)
- Normalisation des noms (slug) compatible fichiers images
- Site statique optimisé pour la rapidité et le responsive
